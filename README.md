# 🔐 Testes de Sign In e Sign Up (Manual)

Projeto simples para validação manual dos fluxos de login (sign in) e cadastro (sign up).

---

🚀 Tecnologias

- HTML
- CSS
- JavaScript

---

🎯 Objetivo

Testar o funcionamento do login e cadastro de usuários.

---

## 🔍 Testes realizados

- Campos obrigatórios
- Dados válidos e inválidos
- Mensagens de erro
- Comportamento da interface

---

▶️ Como executar

1. Baixar o projeto
2. Abrir o arquivo principal no navegador
3. Testar login e cadastro
